# GT-Recommender

## How to clone the respiotory
- - - -
* To clone the folder:
	* `git clone -b Virtual https://github.com/thinhnguyennt7/GT-Recommender.git` 
* To go to directory
	* `cd GT-Recommender`
* To compile the bash script file
	* `chmod u+x bash_script.sh`
* To execute the bash script file
	* `./bashscript.sh`
